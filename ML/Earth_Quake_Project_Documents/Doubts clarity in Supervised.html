🔹 Linear Regression

Purpose: Predict continuous numeric values.
Example Use Case: Predicting house prices, temperatures, salary, etc.

📌 How it works:

  • It tries to draw a straight line (best fit line) through the data points.

  • The goal is to minimize the error between the predicted values and actual values.

  • Formula:

    y=mx+by = mx + by=mx+b

    where:

    • y is the predicted value

    • x is the input feature

    • m is the slope

    • b is the intercept

🔸 Logistic Regression

Purpose: Predict categorical values (mostly binary – 0 or 1).
Example Use Case: Spam detection (spam or not), loan approval (yes or no), disease detection (positive or negative).

📌 How it works:

  • Instead of drawing a straight line, it uses the logistic (sigmoid) function to output values between 0 and 1.

  • These outputs are then interpreted as probabilities.

  • Formula:

    y=11+e−(mx+b)y = \frac{1}{1 + e^{-(mx + b)}}y=1+e−(mx+b)1​

    • Output closer to 1 → class = 1

    • Output closer to 0 → class = 0

🧠 What is a Kernel Function in SVM?

A kernel function transforms your data into a higher-dimensional space where it becomes easier to classify with a hyperplane (a decision boundary).

You don’t manually transform data — the kernel does it implicitly using math tricks. This is called the "kernel trick".

🔹 1. Linear Kernel

  • Definition: No transformation. Just computes the dot product of two vectors.

  • Formula:

    K(x,y)=x⋅yK(x, y) = x \cdot yK(x,y)=x⋅y

  • Use Case: When your data is linearly separable (you can draw a straight line or plane to separate classes).

✅ Output:

  • A real number showing how aligned two vectors are.

  • Higher output → more similar → more likely in the same class.

🔸 2. Polynomial Kernel

  • Definition: Adds polynomial terms to the dot product to allow curved decision boundaries.

  • Formula:

    K(x,y)=(x⋅y+c)dK(x, y) = (x \cdot y + c)^dK(x,y)=(x⋅y+c)d

    where c is a constant (usually 1) and d is the degree of the polynomial.

  • Use Case: When your data isn’t linearly separable but can be separated with curved boundaries.

✅ Output:

  • A real number indicating similarity in polynomial space.

  • Higher output → vectors are similar in polynomial space.

🌌 3. RBF Kernel (Radial Basis Function / Gaussian Kernel)

  • Definition: Measures similarity based on distance — maps data to infinite dimensions!

  • Formula:

    K(x,y)=exp⁡(−γ∥x−y∥2)K(x, y) = \exp\left(-\gamma \|x - y\|^2\right)K(x,y)=exp(−γ∥x−y∥2)

    where γ is a scaling parameter, and ‖x − y‖² is the Euclidean distance squared.

  • Use Case: When data is not linearly separable and you don’t know the right polynomial degree.

✅ Output:

  • A value between 0 and 1

    • 1 → x and y are very similar

    • 0 → x and y are far apart

Activation functions introduce non-linearity into neural networks, helping the model learn complex patterns.

In machine learning models, MSE (Mean Squared Error) and R² (R-squared) are commonly used in regression tasks to evaluate model performance. However, when we talk about accuracy, we usually refer to classification problems. Here's how accuracy is typically used in different types of models:

🧠 Regression Models:

  1. Mean Squared Error (MSE) and R² are used for evaluating how well a regression model predicts continuous values.

    • MSE: Measures the average squared difference between predicted and actual values. Lower MSE means better predictions.

    • R² (Coefficient of Determination): Measures the proportion of variance in the dependent variable that is predictable from the independent variable(s). R² ranges from 0 to 1, where 1 means perfect predictions.

🎯 Classification Models:

In classification models, accuracy is commonly used to evaluate how well the model classifies data into categories. The models that use accuracy as a metric include:

1. Logistic Regression:

  • Use Case: Binary or multi-class classification.

  • Accuracy: Proportion of correctly predicted instances (both 0 and 1 or multiple classes).

Formula for Accuracy:

Accuracy=Number of Correct PredictionsTotal Number of Predictions\text{Accuracy} = \frac{\text{Number of Correct Predictions}}{\text{Total Number of Predictions}}Accuracy=Total Number of PredictionsNumber of Correct Predictions​

2. Support Vector Machine (SVM):

  • Use Case: Classification (binary or multi-class).

  • Accuracy: Measures the percentage of correct classifications.

3. Decision Trees:

  • Use Case: Classification (binary or multi-class).

  • Accuracy: Measures the proportion of correctly classified instances.

4. K-Nearest Neighbors (KNN):

  • Use Case: Classification.

  • Accuracy: Measures the percentage of correct predictions based on neighbors' votes.

✅ Summary:

  • Regression Models (like Linear Regression):

    • Use MSE and R² to evaluate model performance. Accuracy is not used because the output is continuous.

  • Classification Models (like Logistic Regression, SVM, Decision Trees, KNN):

    • Use accuracy as a performance metric, as it deals with categorical outputs (labels).
